# 🎮 스테이지 완전 공략 가이드

## 📊 스테이지 구성 (1-6)

### Stage 1: Bug1 Tutorial (★☆☆☆☆☆)
**목표**: 기본 추적 알고리즘 이해하기

- **적**: Bug1 × 2
- **맵**: 간단한 L자 장애물
- **전략**:
  - Bug1은 벽을 만나면 한 바퀴 돌아야 함
  - 장애물 주변으로 유인 후 반대편으로 도주
  - 대시를 아껴서 사용

**배울 점**: 가장 기본적인 회피 기술

---

### Stage 2: Bug Variations (★★☆☆☆☆)
**목표**: 다양한 Bug 알고리즘 구분하기

- **적**: Bug1, Bug2, Tangent Bug
- **맵**: 여러 장애물
- **전략**:
  - Bug2는 직선 경로를 고집 → 지그재그 이동
  - Tangent Bug는 코너를 잘 돎 → 넓은 공간으로 유도
  - 3마리를 한 곳에 몰아넣지 말 것!

**배울 점**: 각 알고리즘의 행동 패턴 차이

---

### Stage 3: APF Introduction (★★★☆☆☆)
**목표**: 로컬 미니멈 활용 마스터

- **적**: Bug2, APF × 2
- **맵**: U자 구조 (APF 함정용!)
- **전략**:
  - **핵심**: E키로 U자/O자 구조 만들기
  - APF 적을 장벽 안으로 유인
  - 로컬 미니멈에 갇힌 APF는 무력화됨
  - Bug2는 일반 회피로 처리

**배울 점**: 알고리즘 약점 공략의 시작

---

### Stage 4: Graph-Based Planning (★★★★☆☆)
**목표**: PRM/RRT의 경로 차단하기

- **적**: Tangent Bug, PRM, RRT, APF
- **맵**: 복잡한 미로 (여러 방)
- **전략**:
  - PRM 적은 로드맵을 사용 → 경로 차단 효과적
  - E키로 장벽 설치 시 PRM이 재계산 필요 (딜레이 생김)
  - RRT는 자주 재계산하지만 막힐 수 있음
  - 좁은 통로에서 1:1 상대

**배울 점**: 
- PRM/RRT 시각화 관찰 (파란색 선)
- 그래프 기반 알고리즘의 장단점

---

### Stage 5: Probabilistic Tracking (★★★★★☆)
**목표**: Belief 적 교란하기

- **적**: Bug2, APF, Belief × 2, RRT
- **맵**: 기둥이 많은 맵 (시야 차단)
- **전략**:
  - **Q키 (노이즈 폭탄)** 활용!
  - Belief 적(보라색)은 확률 기반 추적
  - 벽 뒤에 숨으면 추적 정확도 하락
  - 노이즈 폭탄 쿨타임 10초 - 신중히 사용
  - 기둥 사이를 지나다니며 혼란 유도

**배울 점**:
- Belief 히트맵 관찰 (보라색 열기)
- 센서 노이즈와 불확실성 이해

---

### Stage 6: BOSS BATTLE 🏆 (★★★★★★)
**목표**: 모든 알고리즘을 동시에 상대!

- **적**: 전부! Bug1, Bug2, Tangent, APF, PRM, RRT, Belief × 2
- **맵**: 중앙 아레나 + 외곽 장애물
- **전략**:
  1. **초반**: 외곽에서 시작, APF부터 무력화
     - E키로 장벽 설치 → APF 로컬 미니멈 유도
  
  2. **중반**: Bug 계열 회피하며 열쇠 수집
     - 대시 활용한 빠른 이동
     - 한 곳에 몰리지 않기
  
  3. **후반**: Belief와 Sampling-based 처리
     - Q키 노이즈 폭탄 (Belief 교란)
     - 장벽으로 PRM/RRT 경로 차단
  
  4. **탈출**: 모든 열쇠 → 중앙 아레나 → 출구

**보스전 팁**:
- 모든 스킬 쿨타임 관리 필수
- 체력 3개 - 2번까지 실수 허용
- 적들을 분산시켜 1:1 상황 만들기
- 장벽은 방어가 아니라 **함정**으로 사용

**배울 점**: 
- 모든 알고리즘의 강점과 약점 종합 이해
- Path-Planning의 실전 응용

---

## 🎓 알고리즘별 약점 요약

| 알고리즘 | 색상 | 약점 | 대응법 |
|---------|------|------|--------|
| Bug1 | 연한 빨강 | 복잡한 장애물에서 시간 낭비 | 장애물 주변 유인 |
| Bug2 | 빨강 | M-line 밖에서 효율 하락 | 지그재그 이동 |
| Tangent Bug | 주황 | 시야 범위 제한 | 시야 밖 이동 |
| APF | 진한 빨강 | **로컬 미니멈** | U자 함정 설치 |
| PRM | 파랑 | 그래프 경로 차단 시 재계산 | 장벽으로 경로 끊기 |
| RRT | 하늘색 | 복잡한 구조에서 헤맴 | 미로로 유도 |
| Belief | 보라 | 센서 노이즈, 시야 차단 | 노이즈 폭탄 + 벽 뒤 숨기 |

## 💡 고급 테크닉

### 1. 장벽 설치 마스터
```
Pattern 1: U자 함정 (APF용)
██████
█    █
█    █

Pattern 2: 일자 차단 (PRM/RRT용)
█████

Pattern 3: L자 유도
███
  ███
```

### 2. 대시 타이밍
- ❌ 안전할 때 사용 (낭비)
- ✅ 적 2마리 이상 접근 시
- ✅ 로컬 미니멈 함정 후 도주
- ✅ 열쇠 수집 후 빠른 이탈

### 3. 노이즈 폭탄 최적화
- Belief 적이 2마리 이상일 때 한 번에 사용
- 쿨타임 10초 - Stage 5, 6에서 2~3번 사용 가능
- 사용 후 즉시 벽 뒤로 숨기

### 4. 미니맵 활용
- 우하단 미니맵으로 적 전체 위치 파악
- 열쇠 위치 확인 → 안전한 경로 선택
- 적들이 한쪽에 몰려있을 때 반대편 공략
